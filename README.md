Чтобы запустить сервер, выполните следующие:

### Установите сначала
### `npm i`

### Запустите REACT в ../client/
### `npm run start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Запустите сервер в ../client/
### `npm run server`
